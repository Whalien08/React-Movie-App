# 🎬 React Movie App

A dynamic movie search and watchlist web application built with React, Vite, and the TMDB API. 

## 🚀 Features
- Fetches real-time movie data (popular listings) from TMDB.
- Dynamic movie search capability.
- Responsive movie card grid layouts.
- Fully styled custom UI components.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, HTML5, CSS3, JavaScript (ES6+)
- **API:** The Movie Database (TMDB) API

## 🔧 Local Setup & Installation

1. **Clone the repository:**
```bash
   git clone <your-repository-url>
   cd frontend
```
2. **Install dependencies:**

Bash
```
   npm install
```

3. **Set up Environment Variables:**

   Create a .env file in the root directory.
```bash
        Add your TMDB API Key: VITE_TMDB_API_KEY=your_actual_api_key
```

4. **Run the development server:**
```bash
   npm run dev
```
